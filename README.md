# MatesX-mini
#### 移动端实时对话 | 无需服务器 | 全网最简部署

---

## 📖 项目简介

Notes：此案例对接阿里百炼平台的 ASR（语音识别）、LLM（大语言模型）和 TTS（语音合成），数字人形象驱动和 VAD（语音活动检测）皆为网页端实时计算。

本项目提供全网最简单的全流程数字人部署代码，无需繁琐的服务器配置，仅需一个 API-Key 即可开启实时对话。

---

## 🚀 快速开始

1. 直接访问[https://kleinlee.github.io/character.html](https://kleinlee.github.io/character.html)。
2. 填入阿里百炼平台的api-key，点击开始，即可与数字人进行实时交互。
3. 克隆项目进行本地修改 
```bash
git clone https://github.com/kleinlee/kleinlee.github.io.git
```

## 🛠️ 技术架构

* 前端计算：VAD 检测、数字人唇形同步与表情驱动均在浏览器端完成，保护隐私，降低延迟。
* 云端服务：对接阿里百炼平台，提供高精度的 ASR、智能 LLM 对话及自然流畅的 TTS 服务。
* 部署方式：支持纯静态部署。
